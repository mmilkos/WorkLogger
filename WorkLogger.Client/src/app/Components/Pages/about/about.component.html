Getting started with work logger

1. Use the login credentials provided in my resume to access the application.

2. The navigation bar at the top provides easy access to different sections of the application. Use this to explore the features.

3. Adding New Users: Navigate to the "Users" section to add new users to the system.

4. Forming Teams: Create teams and assign users to them in the "Teams" section.

5. Creating Tasks: Create new tasks to track your work hours and progress.

6. Managing Task Details: View and update task details, including logging hours spent on each task.

7. Accessing Reports: Go to the "Reports" tab to generate excel reports for each team. You can customize the reports by selecting the desired time period.
